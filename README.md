# Breast_Cancer



Le cancer du sein est une maladie où la détection précoce améliore fortement les chances de survie.   Les méthodes traditionnelles (biopsies) sont lentes et coûteuses, d'où l'intérêt du Machine Learning (ML) et du Deep Learning (DL) pour analyser rapidement les caractéristiques cellulaires issues de la méthode FNA.
